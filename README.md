# Instagen

Instagen is a web-based tool that allows users to generate **Instagram-style mockups** including profiles, posts, and comments. It is designed to help creators, marketers, and developers quickly create realistic-looking Instagram previews that can be **downloaded and shared**.

The goal of Instagen is to make mockup creation simple, fast, and accessible without needing complex design software.
